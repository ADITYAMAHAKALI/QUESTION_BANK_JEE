# Question
Which of the following compounds undergoes nucleophilic substitution ($S_N1$) fastest?

# Options
- **A**: tert-Butyl chloride
- **B**: Isopropyl chloride
- **C**: Ethyl chloride
- **D**: Methyl chloride

# Explanation
$S_N1$ reactions proceed via a carbocation intermediate. tert-Butyl chloride forms a stable $3^\circ$ carbocation, which is highly stabilized by hyperconjugation and inductive effect, making it react the fastest.
