# Question
The spin-only magnetic moment of $[\text{Fe}(\text{H}_2\text{O})_6]^{2+}$ is approximately:

# Options
- **A**: $4.90\text{ BM}$
- **B**: $5.92\text{ BM}$
- **C**: $2.83\text{ BM}$
- **D**: $0\text{ BM}$

# Explanation
In $[\text{Fe}(\text{H}_2\text{O})_6]^{2+}$, Fe is in $+2$ oxidation state ($d^6$). Since $\text{H}_2\text{O}$ is a weak field ligand, no pairing occurs. Number of unpaired electrons $n = 4$. Spin-only magnetic moment $\mu = \sqrt{n(n+2)} = \sqrt{4(6)} = \sqrt{24} \approx 4.90\text{ BM}$.
