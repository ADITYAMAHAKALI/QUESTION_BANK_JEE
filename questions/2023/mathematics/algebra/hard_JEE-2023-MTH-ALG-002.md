# Question
If $A$ is a $3 \times 3$ non-singular matrix such that $A A^T = A^T A$ and $B = A^{-1} A^T$, then $B B^T$ is equal to:

# Options
- **A**: $I$ (Identity Matrix)
- **B**: $A$
- **C**: $B$
- **D**: $A^{-1}$

# Explanation
$B B^T = (A^{-1} A^T) (A^{-1} A^T)^T = A^{-1} A^T A (A^{-1})^T = A^{-1} A A^T (A^T)^{-1} = I \cdot I = I$. (Using the property that $(A^T)^{-1} = (A^{-1})^T$ and $A^T A = A A^T$).
