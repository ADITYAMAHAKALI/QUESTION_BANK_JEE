# Question
If $\omega$ is a complex cube root of unity, then the value of $(1 - \omega + \omega^2)^5 + (1 + \omega - \omega^2)^5$ is:

# Options
- **A**: $-32$
- **B**: $32$
- **C**: $64$
- **D**: $-64$

# Explanation
We know $1 + \omega + \omega^2 = 0$. So $1 + \omega^2 = -\omega$ and $1 + \omega = -\omega^2$. Thus, $(1 - \omega + \omega^2)^5 = (-2\omega)^5 = -32\omega^2$ (since $\omega^5 = \omega^2$). Similarly, $(1 + \omega - \omega^2)^5 = (-2\omega^2)^5 = -32\omega^{10} = -32\omega$. The sum is $-32(\omega^2 + \omega) = -32(-1) = 32$.
