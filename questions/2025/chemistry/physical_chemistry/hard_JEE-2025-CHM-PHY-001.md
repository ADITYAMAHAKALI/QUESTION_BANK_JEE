# Question
The standard electrode potentials for $\text{Cu}^{2+}/\text{Cu}^+$ and $\text{Cu}^+/\text{Cu}$ are $0.15\text{ V}$ and $0.50\text{ V}$ respectively. The value of $E^\circ$ for $\text{Cu}^{2+}/\text{Cu}$ is:

# Options
- **A**: $0.325\text{ V}$
- **B**: $0.65\text{ V}$
- **C**: $0.35\text{ V}$
- **D**: $0.175\text{ V}$

# Explanation
Using Gibbs free energy relations: $\Delta G^\circ = \Delta G_1^\circ + \Delta G_2^\circ \implies -n F E^\circ = -n_1 F E_1^\circ - n_2 F E_2^\circ$. Here $n = 2$, $n_1 = 1$, $n_2 = 1$. Thus, $2 E^\circ = 1(0.15) + 1(0.50) = 0.65 \implies E^\circ = 0.325\text{ V}$.
